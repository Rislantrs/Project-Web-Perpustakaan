# Perpustakaan Digital - Dinas Arsip dan Perpustakaan

A modern, cozy, and professional library website designed for government archive and library services. Built with React, TypeScript, Tailwind CSS, and inspired by design systems from Elsevier and the New York Public Library.

## 🎨 Design System

### Color Palette
- **Primary**: Deep Navy (#1E3A5F) - Professional and trustworthy
- **Secondary**: Light Gray (#F5F5F5) - Clean and minimal
- **Accent**: Warm Beige (#C8A97E) - Cozy and inviting
- **Background**: Soft White (#FFFFFF)
- **Sage Green**: (#9DAD97) - Calm accent color

### Typography
- **Headings**: Playfair Display (serif, elegant)
- **Body**: Inter (clean, modern sans-serif)
- **Cultural Accents**: Noto Sans Sundanese (for cultural elements)

### Design Principles
- Minimalist and elegant
- Warm and cozy reading atmosphere
- Academic and trustworthy feel
- Lots of whitespace
- Smooth transitions and micro-interactions
- Card-based layouts with soft shadows
- Rounded corners throughout

## 🚀 Features

### Pages
1. **Homepage (Beranda)**
   - Hero section with smooth animations
   - Search bar for books, articles, and archives
   - Popular books carousel
   - Latest articles section
   - Upcoming events preview
   - Call-to-action for member registration

2. **About Page**
   - Institution profile
   - Vision & Mission (2-column layout)
   - Values and principles
   - Sundanese cultural quote section

3. **Organization Page**
   - Interactive organizational chart
   - Expandable/collapsible hierarchy
   - Click to view position details
   - Modal with contact information

4. **Book Catalog**
   - Grid layout with book cards
   - Advanced filters (category, year, search)
   - Sorting options (popularity, rating, year, title)
   - Book detail pages with reviews
   - Availability status
   - Borrow functionality

5. **Articles Page**
   - Academic-style article list
   - Category filters
   - Download PDF functionality
   - Author and date information

6. **Digital Archive**
   - Archive document grid
   - Filter by type and year
   - PDF preview viewer
   - Download functionality

7. **Events Page**
   - Calendar view integration
   - Event cards with registration status
   - Category indicators
   - Event detail pages with agenda

8. **Member System**
   - Registration form
   - Login page
   - User dashboard with:
     - Currently borrowed books
     - Borrowing history
     - Favorite books
     - Statistics

## 🎯 UX Features

- **Smooth Animations**: Fade-in, slide, and scale animations using Framer Motion
- **Hover Effects**: Card scaling and shadow effects on hover
- **Responsive Design**: Mobile-first approach, works on all devices
- **Dark Mode**: Cozy reading mode for comfortable nighttime browsing
- **Micro-interactions**: Button feedback, loading states, toast notifications
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation

## 🛠 Tech Stack

- **Framework**: React 18 with TypeScript
- **Routing**: React Router v7 (Data mode)
- **Styling**: Tailwind CSS v4
- **Animations**: Motion (Framer Motion)
- **UI Components**: Radix UI
- **Carousel**: React Slick
- **Theme**: next-themes (dark mode support)
- **Notifications**: Sonner

## 📱 Responsive Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px
